# Romantic Valentine Proposal Website

A beautiful, interactive Valentine's Day proposal website with engaging animations and playful UI elements.

## Features

- **Envelope Intro Animation** - Floating love letter with wax seal that opens on click
- **Typewriter Effect** - Text types out character by character
- **Starry Night Background** - Twinkling stars with shooting stars
- **Heart Cursor Trail** - Hearts follow your mouse movement
- **Escaping "No" Button** - Button runs away when you try to click it
- **Love Meter** - Animated progress bar
- **Floating Hearts & Rose Petals** - Romantic particle effects
- **Confetti Celebration** - Bursts when "Yes" is clicked
- **Fully Responsive** - Works on desktop and mobile

## Demo

Simply open `index.html` in any modern browser.

## Customization

### Change Background Image
Edit line 120 in `index.html`:
```css
background-image: url('your-image.jpg');
```

### Change Title Text
Edit line 941 in `index.html`:
```javascript
typeWriter("Your Custom Message Here", titleEl);
```

### Change Subtitle Quote
Edit line 781 in `index.html`:
```html
<p class="subtitle">"Your romantic quote here"</p>
```

## Technologies Used

- HTML5
- CSS3 (Animations, Flexbox, Grid)
- Vanilla JavaScript
